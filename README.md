# Initial Setup
`npm install`

# Execution
## With default params
`npm start`

## With params
`npm start -- --input i.txt --ouput o.txt`

## Params
- `--input, -i` : output file path (default: `"input.txt"`)
- `--output, -o` : output file path (default: `"output.txt"`)
- `--debug, -d` : print debug messages (default: `false`)

# Clean
`npm run clean`

# Test
`npm test`
